About these protocols

Some more text describing how incredible and amazing this website is and how it will change forever the face of parasite research.

Authors

  • The nemabiome concept and development of tools to study the nemabiome is done in the lab of John Gilleard
  • Bench protocols written by Elizabeth Redman
  • Analysis protocols written by Russell Avramenko
  • ITS2 database and taxonomy constructed by Russell Avramenko
  • Website built by Matthew Workentine

*This website is hosted on github.com, code available here

Citation

If you use any of the described protocols in your work we kindly ask that you credit this website as well as cite the following journal article:

Avramenko RW, Redman EM, Lewis R, Yazwinski TA, Wasmuth JD, Gilleard JS. Exploring the Gastrointestinal “Nemabiome”: Deep Amplicon Sequencing to Quantify the Species Composition of Parasitic Nematode Communities. PLoS ONE. 2015;10: e0143559. doi: 10.1371/journal.pone.0143559 PDF
